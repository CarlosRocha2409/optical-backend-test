# Optical Backend Test

## Env Variables

This variables are necesary for starting the server up

```
//Server port
PORt=
//Database connection string
DB_CONNECTION=
//Database user
DB_USER=
//Database port
DB_PORT=
//Database password
DB_PASSWORD=
//Database name
DB=
```

## Scripts

### Start development server

```bash
$ yarn dev
```

### Start production server

```bash
$ yarn start
```
